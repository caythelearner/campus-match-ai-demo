## 1. 必须准备

### Python 环境

建议使用 Python 3.10 或 3.11。

```bash
cd campus_match_ai
python3 -m venv .venv
source .venv/bin/activate
pip install numpy pillow requests streamlit networkx neo4j
```

Windows PowerShell 如果不用 WSL，可改成：

```powershell
cd campus_match_ai
py -3.10 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install numpy pillow requests streamlit networkx neo4j
```

### 跑通默认全链路

```bash
python scripts/run_pipeline.py --n-users 100 --top-k 5
streamlit run app.py
```

如果只想快速检查代码是否能跑，先用小规模数据：

```bash
python scripts/run_pipeline.py --n-users 30 --top-k 5
```

浏览器打开 Streamlit 后，至少确认这些页面内容存在：

1. 用户画像。
2. Top 匹配推荐。
3. GraphRAG 路径证据。

## 2. Neo4j 准备

Neo4j 不是前端 demo 的硬依赖。即使不装 Neo4j，pipeline 也会自动生成可提交的图数据库留痕文件：

```text
outputs/neo4j/campus_match_ai_nodes.csv
outputs/neo4j/campus_match_ai_relationships.csv
outputs/neo4j/import_campus_match_ai.cypher
outputs/neo4j/demo_queries.cypher
outputs/neo4j/neo4j_trace_summary.json
```

如果想在本机真的打开 Neo4j 截图，提前安装 Neo4j Desktop 或 Neo4j Community，创建一个本地数据库，记住密码，然后执行：

```bash
python scripts/import_neo4j.py --uri bolt://localhost:7687 --user neo4j --password 你的密码 --clear
```

Neo4j Browser 里优先跑这个截图：

```cypher
MATCH p=(u:User {node_id: 'U001'})-[r]->(x)
RETURN p
LIMIT 80;
```

更多查询在：

```text
outputs/neo4j/demo_queries.cypher
```

如果没有时间安装 Neo4j，也可以直接在 Streamlit 的“Neo4j 图数据库留痕”区截图，说明这是默认生成的图数据库交付证据。

## 3. 可选增强

### BGE/CLIP 语义增强

3070 Ti 可以跑 BGE/CLIP 推理，但首次下载模型可能比较慢。

```bash
pip install sentence-transformers transformers torch
```

然后把 `configs/default.json` 里的 embedding 配置改成：

```json
{
  "text_provider": "sentence_transformer",
  "text_model_name": "BAAI/bge-small-zh-v1.5",
  "image_provider": "clip",
  "clip_model_name": "openai/clip-vit-base-patch32"
}
```

不建议在 3070 Ti 上做大模型微调；A100 那边再跑 GraphSAGE 或 LoRA。

### 图片生成 API

如果要把占位图替换成更像样的合成生活方式图，提前准备图像生成 API：

```bash
export IMAGE_API_URL="你的图像生成接口地址"
export IMAGE_API_KEY="你的 API key"
export IMAGE_API_MODEL="你的模型名"
export IMAGE_API_RESPONSE_MODE="url"
```

然后把 `configs/default.json` 的 `generation.image_provider` 改成 `api`。
